# ⚡ Getting Started

## Clone Repository

```bash
git clone https://github.com/Aytech-1/aytech-portfolio.git
```

Move into the project:

```bash
cd your-portfolio
```

---

## Install Dependencies

```bash
npm install
```

or

```bash
yarn install
```

---

## Start Development Server

```bash
npm run dev
```

Open:

```
http://localhost:3000
```

---

# 🔐 Environment Variables

Create:

`.env.local`

```env
NEXT_PUBLIC_SITE_URL=http://localhost:3000

WEB3FORMS_ACCESS_KEY=YOUR_WEB3FORMS_ACCESS_KEY
```
